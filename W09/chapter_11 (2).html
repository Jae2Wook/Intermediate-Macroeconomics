Preliminaries: Putting the ISLM Curve into Python¶

The IS curve is a simple linear function! However, incorporating the formula $\tilde{Y_t}=\bar{a}-\bar{b}(R_t-\bar{r})$ into Python can be a little tricky because $R_t$ is the dependent variable, not the independent variable when we are graphing it on the Carthesian plane. If I was doing this homework, these are the steps I would take to graph the IS curve:

  • Solve for $R_t$ in this equation $\tilde{Y_t}=\bar{a}-\bar{b}(R_t-\bar{r})$.
  • Create an array that contains the range of $\tilde{Y}_t$: $\left(-\bar{a}-\bar{b}\bar{r}, \;\bar{a}+\bar{b}\bar{r}\right)$.
  • Use the formula mentioned in bulletpoint 1 to calculate the corresponding values for $R_t$.
  • Graph it! Use the appropriate labeling of values of the x-axis so it looks right.

R_t = (a - Y_st) / b + r\ Y_st = [-a-br, a + br] range\ y-axis: real interest rate(Rt), x-axis: Y_st

Q4) Consumption and the Real Interest Rate¶

According to the life-cycle/permanent-income hypothesis, consumption depends on the present dicounted value of income. An increase in the real interest rate will make future income worth less, thereby reducing the present discounted value and reducing consumption. To incorporate this channel into the model, suppose the consumption equation is given by

\begin{align*} C_t = \bar{a_c}\bar{Y_t}-\bar{b_c}(R_t-\bar{r})\bar{Y_t} \end{align*}

Assume the remainder of the model is unchanged from the original setup, as in Table 11.1 (pg. 283)

(a) Derive and graph the IS curve for this new specification. Write your answers below

Y_st = a -(b + b_c) * (Rt - r)\ Rt = ((a - Y_st) / (b + b_c)) + r\ becoming flatter slope because b became bigger

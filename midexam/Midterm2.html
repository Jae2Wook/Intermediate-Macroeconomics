## This is the Phase Diagram from class, showing "per capita steady-state"

import numpy as np
import matplotlib.pyplot as plt

s=0.1
d= 0.06
a= 0.33
l= 0.01

g= g_avg
n= n_avg

k= np.linspace (0,5, num= 1000)

Dep= []
Inv= []
Out= []

for i in range (len(k)):
    D= (d+g+n)*k[i]
    Dep.append(D)
for i in range(len(k)):
    I= s*((1-l)**(1-a))*(k[i]**a)
    Inv.append(I)
for i in range(len(k)):
    Y= (1-l)**(1-a)*(k[i]**a)
    Out.append(Y)

print("capital per capita (k*): 1.466")
print("output per capita (y*):1.105")
print()

plt.plot(k,Dep, c="firebrick", label= "Effective Depreciation")
plt.plot(k,Inv, c="steelblue", label= "Effective Investment")
plt.title("Romer and Solow Model- Phase Diagram")
plt.axvline(x=1.38, c= "grey", linestyle="--", label= "Steady State")
plt.xlabel("Capital per capita")
plt.ylabel("Output per capita")
plt.legend()
plt.show()

## Using Romer/Solow to show Output, Capital, and Consumption: 

import numpy as np
import matplotlib.pyplot as plt


#Step 1: Define the production function that combines Romer/Solow
def production_func (l, A, K, L, a):
    return (((1 - l) ** (1 - a)) * (A ** (1 - a)) * (K ** a) * (L ** (1 - a)))

#Step 2: Define the inital variables
K0 = 1000
L0 = 500
A0 = 1.7

#Step 3: Define our variables as arrays
Capital = [K0]                                       #big K= Capital
Labor = [L0]                                         #L= Labor
TFP = [A0]                                           #TFP= Knowledge
Output = [production_func(l, A0, K0, L0, a)]
Consumption = [(1-s) * Output[0]]                    #Consumption= (1-s)*Y [page 7: Moncayo's notes]
little_k = [K0 / (A0 * L0)]                          #little k = capital per person

#Step 4: Define time and periods: 
t = 0
period = [t]


#Step 5: Set up loop:

while abs(little_k[t] - 1.466) > .01:
    Capital= np.append(Capital, Capital[t] + (s * Output[t] - d * Capital[t]))
    Labor= np.append(Labor, Labor[t] * (1 + n))
    TFP= np.append(TFP, TFP[t] * (1 + g))
    Output= np.append(Output, production_func(l, TFP[t+1], Capital[t+1], Labor[t+1], a))
    Consumption= np.append(Consumption, (1-s)*Output[t+1])
    little_k= np.append(little_k, Capital[t+1] / (TFP[t+1] * Labor[t+1]))
    t += 1
    period.append(t)
    
#Step 6: Print values from question:
print("Time needed to reach steady state: {} years" .format(t))
print()
print("Capital at steady state: {:.2f}" .format(Capital[-1]))
print("Labor at steady state: {:.2f}" .format(Labor[-1]))
print("TFP at steady state: {:.2f}" .format(TFP[-1]))
print()


#Step 7: Print graph:
plt.plot(period, Capital,c= "firebrick", label = "Capital")
plt.plot(period, Output, c="darkolivegreen", label = "Output")
plt.plot(period, Consumption, c= "steelblue", label = "Consumption")
plt.axvline(x=t, color = "dimgrey", linestyle = "--", label = " Steady-state")
plt.title("Growth Path of: Output, Consumption, and Capital")
plt.yscale('log')
plt.xlabel("Time in Years")
plt.ylabel("Value in Dollars")
plt.legend()

5) For each of the previous scenarios, please discuss what you found. Include details about the length of time it took to reach the steady-state of k. Was there any change in the steady-state? What do you think are the welfare consequences of each scenario occurring in real life?¶

$k_t^*$ $=$ $(1-{\ell})$$(\frac{s}{\delta + g + n})^{\frac{1}{1-a}}$

In each of the four scenarios, at the 10 year period some sort of shock happened. The effective steady state (k*) is a function of population growth(n), technoligcal growth(n), savings rate(s) and depreciation(d). So unless one of these factors change, the steady state should remain unchanged, with only the time to reach such steady-state fluctuating depending on the shock. Welfare consequences in real life are not going to change unless the steady-state changes. So if the steady-state remains the same there are no effects on welfare.

In the first scenario where war destroys half the capital in the US, capital drops significantly at 10 year mark, while output and consumption both decrease but not as sharply. It then takes 97 years to return to steady state, which is unchanged at 1.38 capital per capita. Welfare is unchanged in the long run once the economy has returned back to the steady-state.

In the second scenario the savings rate doubles. I was not able to make a graph for this one, however capital and output should increase. While consumption should decrease if people are saving more. The steady state amount of capital per person will increase becuase its a function of savings, and this would then raise the welfare consequences making everyone better off.

In the third scenario technology instantly doubles. This shifts all three; capital, output, and consumption, higher intitally at the 10 year mark. However, in 98 years the growth (of Kt, Ot,Ct) returns to a linear growth path and capital per capita is unchanged. Therefore not effecting the welfare.

Finally in the fourth scenario, a quarter of the workforce leaves. This initally decreases capital, output, and consumption of course at the 10 year mark. But after 70 years we reach the orginal steady state again. In the long-run again not effecting the welfare of individuals.

These comparative statics, support that the only way to acutally change welfare in real life is to change the population growth rate, technolical growth rate, the savings rate, or depreciation. Any changes to those rates would then change welfare through the shift of the steady state values.
